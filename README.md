# pifu.me

A personal website built with [Hugo](https://gohugo.io/) and styled using [missing.css](https://missing.style/).

## Usage

**Make sure to install [Hugo](https://gohugo.io/) first.**

To run locally:

```sh
git clone https://github.com/doukeshi/pifu.me.git
cd pifu.me
make serve
```

This will start a local development server using Hugo.
